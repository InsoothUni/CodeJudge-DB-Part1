# CodeJudge Database System - Part 1

This repository contains Part 1 of the SQL & DBMS Assignment: Relational Design, Keys & Normalization.

### Included Files:
* `schema.sql`: The complete DDL statements for the database schema.
* `schema_explanation.md`: Breakdown of entities and the overall design.
* `keys_and_relationships.md`: Justification for primary, foreign, and composite keys.
* `normalization_notes.md`: Analysis of redundancy and 3NF design.
* `erd.md`: A Mermaid text-based Entity Relationship Diagram.
* `assumptions.md`: Key assumptions made while interpreting the raw dataset.
